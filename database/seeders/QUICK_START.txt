🚀 HOW TO USE (3 SIMPLE STEPS):

STEP 1: PLACE YOUR EXCEL FILE
──────────────────────────────
Copy your Excel file to this location:
  📁 database/import/la_base_de_donnes_de_nouveau_systemes_18-05.xlsx

Your Excel should have these columns:
  • CODE        - Product code
  • ARTICLES    - Product name
  • FRS         - Supplier name
  • CATEGORIE   - Category
  • FAMILLE     - Family
  • S FAMILLE   - Sub-family
  • UNITEE      - Unit (kg, litre, pièce, etc.)
  • PRIX UN     - Unit price
  • TVA         - Tax percentage
  • QUANTITE    - Quantity

STEP 2: PREVIEW THE DATA (Optional but Recommended)
────────────────────────────────────────────────────
Run this to see what will be imported:
  
  php database/seeders/PreviewExcelImport.php

This shows:
  • Number of products in each sheet
  • List of suppliers
  • Categories and families
  • Units found
  • Total inventory value
  • Sample products

STEP 3: IMPORT THE DATA
───────────────────────
Choose ONE of these methods:

METHOD A - Using Artisan Command (EASIEST) ⭐
  php artisan products:import-excel

METHOD B - Using Seeder Directly
  php artisan db:seed --class=ProductsFromExcelSeeder

METHOD C - Run Setup Check First
  php database/seeders/SetupExcelImport.php

═══════════════════════════════════════════════════════════════════════

📊 WHAT GETS IMPORTED:

For each product in your Excel (all 5 sheets):
  ✅ Product record created
  ✅ Supplier created (from FRS column)
  ✅ Category created (from FAMILLE column)
  ✅ Unit created (from UNITEE column)
  ✅ Barcode created (using CODE)
  ✅ Store-Product link created
  ✅ Purchase order created (validated status)
  ✅ Stock added (from QUANTITE column)
  ✅ Stock movement tracked

Price Calculation:
  • Purchase Price = PRIX UN
  • Selling Price = PRIX UN × (1 + TVA%)
  
Example:
  PRIX UN = 100 MAD, TVA = 20%
  → Purchase Price = 100 MAD
  → Selling Price = 120 MAD
